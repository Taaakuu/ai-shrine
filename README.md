# AI Shrine / 神庙项目

AI Shrine 是一个结合 **AI Agent、数字叙事、交互体验与未来社区想象** 的实验性项目。

这个项目尝试构建一个“数字神庙”：

人们可以在这里提出愿望、记录想法、与 AI 角色互动，并通过技术的方式，把抽象的信念、目标和故事转化为可持续生长的数字空间。

## 项目愿景

AI Shrine 不只是一个普通的网站或工具。

它更像是一个面向未来的精神入口：

* 每个人都可以拥有自己的数字仪式感
* 每个愿望都可以被记录、回应和追踪
* AI 不只是工具，也可以成为陪伴者、守护者和引导者
* 技术不只是效率系统，也可以承载情感、信念和想象力

我们希望通过这个项目探索：

**当 AI 成为新的基础设施之后，人类如何重新设计自己的精神空间、社群关系和数字身份。**

## 核心功能方向

当前项目处于早期探索阶段，计划逐步实现以下功能：

### 1. 愿望记录

用户可以在神庙中写下自己的愿望、目标、困惑或人生问题。

这些内容可以被保存、回顾，并在未来形成个人成长轨迹。

### 2. AI 神庙角色

项目将设计不同类型的 AI 角色，例如：

* 神庙守护者
* 愿望记录员
* 人生引导者
* 任务监督者
* 灵感生成者

这些角色将根据用户输入，提供回应、建议、提醒或陪伴。

### 3. 数字仪式感

项目希望用交互设计营造一种轻量但有意义的仪式感，例如：

* 每日祈愿
* 每周复盘
* 目标确认
* 情绪记录
* 未来信件
* 成长时间线

### 4. Web3 扩展可能

未来项目也可能探索 Web3 方向，例如：

* NFT 身份
* 数字护符
* 链上愿望证明
* 社群成员身份
* 去中心化精神社区

目前 Web3 不是项目的唯一目标，而是未来可能扩展的方向之一。

## 技术方向

项目当前可能涉及以下技术方向：

* Frontend：HTML / CSS / JavaScript / Vue / React
* Backend：Node.js / Python / Java / Spring Boot
* AI：OpenAI API / Agent Framework / Prompt Engineering
* Database：PostgreSQL / MySQL / SQLite

具体技术栈会根据项目推进逐步确定。
